# NeighborFood Frontend 
공동 주문 배달 앱 

<br/>

## 1. 제작 기간
 - 2023.04.25 ~ 2023.06.02 

<br/>

## 2. 실행방법
 1. `npm install --legacy-peer-deps` 실행하여 node_modules 폴더를 다운
 2. `npm start` 실행

<br/>

## 3. 역할
|이름|구현 기능|
|---|---|
|백지원|마이페이지, 내가 참여한 게시물 화면|
|신승미|모임 게시물 관리 화면 및 음식점 조회 화면|
|이혜원|로그인, 회원가입, 메뉴 조회 및 공동 장바구니 관리 화면|

## 4. 사용 기술 및 API
 - CSS
 - React
 - 모피어스 API
  - Location API
  - Data API
 - Material-UI

<br/>
